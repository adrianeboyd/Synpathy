How to edit:

  1. importing terminal nodes (words):

    You have to import terminal nodes from an external plain-text-file. It should have the following block structure for each sentence, followed by an empty line:

    you go out of the institute
    pro v adv prep art n
    Apart from the words, each block can contain an arbitrary number of additional lines, representing further terminal node features like 'Part-of-Speech', etc.



  2. adding/manipulating non-terminal nodes:

    First select at least one node by pressing the left mouse button. You can mark further nodes by keeping the CTRL-key pressed while clicking with the mouse (however you cannot mark nodes beneath another marked node).

    Now, if you press the right mouse button, a popup-menu will be shown from which you can select an editing operation. For adding an egde you first have to select the "higher" node (where the edge comes from), than an arbitrary number of "lower" nodes (where the edge goes to). Secondary edges can point in the opposite direction, too.

    In order to select labels, you first need to open an existing feature configuration (via menu "File") or to define it on you own (menu "Edit").

    To edit the label of a node, double-click into the node. A popup-menu will show. In the case of nonterminal nodes, you have to choose an entry from the offered list. If you need an item not present in the list, add it with the "Edit->Edit ... feature" option (and don't forget to save the changes!). For terminal nodes, you may modify the list entries, but this will affect only the selected node. In order to change a value in the list permanently, you have to choose here again the "Edit" option.
